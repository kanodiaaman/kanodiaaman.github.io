## ✍️ Writing Philosophy

```
❝  Write so the first-timer understands it.
   Write so the expert can verify it.
   Leave no gap between the two.              ❞
                               — @kanodiaaman
```

Every article on this site follows three rules:

- **No domain gaps** — If the spec says it, the article explains it
- **No filler** — Every sentence earns its place
- **No assumptions** — Concepts are built from the ground up, then verified at the top
